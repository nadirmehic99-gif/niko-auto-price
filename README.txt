Niko Auto Core USA - Excel Upload Website

How it works:
1. Open admin.html
2. Login with password: mafija1
3. Upload Excel .xlsx file
4. Excel needs columns: Product and Price
   Also accepted: Serial/Code and Value
5. Open index.html and search serial numbers

Important:
This version saves prices in the browser localStorage.
For all customers online, connect it to Supabase or GitHub so one upload updates everyone.

Files:
- index.html = customer search
- admin.html = admin Excel upload
- style.css
- script.js
- admin.js
- sample-price-list.csv
